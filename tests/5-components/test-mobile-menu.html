This page verifies the functionality of the mobile-menu classes defined in the 5-components Sass layer. The overlay examples below are expected to render only below the layout breakpoint. At desktop and wider viewports, the test frames may appear empty because the component is intentionally disabled there.

Hidden state

Expected behavior: A mobile menu instance using the hidden attribute should not render at all. You should only see the helper preview box below, not the overlay panel itself.

This helper box should remain visible. The hidden mobile menu should not appear.
